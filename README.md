
# 🚀 نظام A-E Storage Ecosystem - الوصف الشامل (Project Blueprint)

نظام إداري ومحاسبي سحابي متكامل، مصمم ليكون الحل النهائي لإدارة المستودعات، الديون، والتدفقات النقدية للمؤسسات الصغيرة والمتوسطة. يتميز النظام بالدقة المتناهية والمزامنة اللحظية بين جميع الأجهزة.

## 🛠️ المواصفات التقنية (Tech Stack)
- **الواجهة الأمامية:** Next.js 15 (App Router) - أحدث إصدار للأداء الفائق.
- **التصميم:** Tailwind CSS مع مكتبة Shadcn UI (تصميم عصري، متجاوب، ويدعم RTL).
- **قاعدة البيانات:** Google Firebase Firestore (سحابية، مشفرة، وتعمل في الوقت الفعلي).
- **إدارة الحالة:** React Hooks & Context API (لسرعة استجابة الواجهة).
- **المحاسبة:** نظام حسابات مدمج يعتمد على Double-Entry Logic لضمان توازن الديون والمخزون.

## 📦 الوحدات والوظائف الأساسية

### 1. الإدارة الذكية للمخزون (Inventory 2.0)
- **تنظيم هرمي:** تقسيم الأصناف إلى (أقسام رئيسية) و(وحدات قياس) مخصصة.
- **تتبع المخزون:** حساب تلقائي للكميات المتاحة، أسعار الشراء (التكلفة)، وأسعار البيع.
- **نظام التنبيه:** مؤشرات بصرية (Badges) تنبهك فوراً عند وصول أي صنف لـ "حد الطلب" (أقل من 10 قطع).

### 2. منظومة الموردين والديون (Debt Management)
- **ملفات مالية:** سجل لكل مورد يشمل إجمالي مشترياته، المبالغ المسددة، وصافي مديونيته.
- **الربط التلقائي:** أي فاتورة مشتريات تزيد ديون المورد وتزيد المخزون في حركة واحدة مؤمنة.
- **سجل السدادات:** توثيق كل دفعة مالية للمورد مع التاريخ وطريقة الدفع (كاش، تحويل، شيك).

### 3. المشتريات وفواتير التوريد (Procurement)
- **فواتير احترافية:** تسجيل فواتير متعددة الأصناف مع حساب الإجمالي، المدفوع، والمتبقي كدين.
- **تحديث المخازن:** بمجرد حفظ الفاتورة، يتم زيادة الكميات في المستودع وتسجيل حركة "وارد" في السجل العام.

### 4. الحوالات والكاش (Cashier Hub)
- **محرك الحوالات:** تسجيل المبالغ المرسلة والمستلمة من الأشخاص خارج نطاق الموردين.
- **البحث المالي:** ابحث باسم أي شخص ليقوم النظام فوراً بحساب (صافي حسابه) بناءً على كل تعاملاته السابقة (مرسل/مستلم).

### 5. المالية العامة والمبيعات (General Finance)
- **المبيعات اليومية:** سجل لإجمالي المبيعات وعدد الفواتير لكل يوم.
- **المصاريف التشغيلية:** تتبع المصاريف (إيجار، رواتب، كهرباء) وتأثيرها على الأرباح.
- **الصافي اليومي:** حساب تلقائي لصافي الربح (المبيعات - المصاريف).

### 6. الأمان وصلاحيات الوصول (Access Control)
- **نظام الأدوار:** 
  - **Admin:** صلاحيات كاملة (إعدادات، حذف، تقارير مالية).
  - **Editor:** إضافة وتعديل البيانات بدون صلاحيات الحذف أو الإعدادات.
  - **Logger:** تسجيل الحركات اليومية فقط.
- **Master Key:** رمز حماية إداري (`abdallah123456a`) للعمليات الحساسة وتصدير البيانات البرمجية.

### 7. التقارير والتحليلات (Business Intelligence)
- **رسوم بيانية:** تمثيل مرئي لحجم المشتريات، توزيع الأصول، والمنتجات الأكثر حركة.
- **تصدير شامل:** استخراج تقارير Excel (7 Sheets)، PDF للطباعة، و JSON للنسخ الاحتياطي.
