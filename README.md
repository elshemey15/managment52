
# نظام A-E Storage Ecosystem

هذا النظام جاهز للنشر السحابي باستخدام تقنيات Next.js و Firebase.

## ملاحظة هامة للنشر:
إذا ظهرت لك صفحة Firebase الترحيبية (Welcome Page) بدلاً من النظام، فهذا يعني أن هناك ملف `index.html` افتراضي تم إنشاؤه. لقد قمنا بضبط الإعدادات لتجاهله.

## خطوات الحصول على الرابط الفعال:
1. تأكد من تفعيل **App Hosting** في لوحة تحكم Firebase.
2. اضغط على زر **Deploy** في شريط أدوات المحرر.
3. انتظر حتى تكتمل عملية البناء (Build) - قد تستغرق دقيقتين لأن نظام Next.js يحتاج لبناء الواجهات برمجياً.
4. افتح الرابط الذي سينتهي بـ `.web.app` وستجد النظام يعمل بكامل طاقته.

**تنبيه:** لا تنقل ملفات الكود إلى مجلد `public`؛ الكود يجب أن يبقى في `src/app` لضمان عمل المزامنة السحابية وتأمين البيانات.
