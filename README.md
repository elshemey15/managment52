
# 🚀 نظام A-E Storage Ecosystem - دليل الحفظ السحابي والرفع على GitHub

هذا النظام مصمم ليكون **خزنة بياناتك الدائمة**. البيانات محفوظة في سحابة جوجل (Firebase)، والكود الخاص بك سيتم حفظه في GitHub.

## 🔐 ضمان حفظ البيانات للأبد
- **لا حاجة للضغط على "حفظ":** النظام يستخدم تقنية المزامنة اللحظية مع **Firebase Firestore**.
- **الأمان:** حتى لو تعطل جهازك، بمجرد فتح الرابط وتسجيل الدخول، ستظهر بياناتك كاملة.

## 🛠 كيفية رفع المشروع على GitHub (للمبتدئين)
إذا كنت تريد حفظ "كود" الموقع على حسابك في GitHub، اتبع الخطوات التالية من الـ CMD:

1. **إنشاء مستودع (Repository) جديد:** اذهب إلى [GitHub](https://github.com/new) وأنشئ مستودعاً جديداً باسم `ae-storage`.
2. **تنفيذ الأوامر التالية في مجلد المشروع على جهازك:**
   ```bash
   git init
   git add .
   git commit -m "إطلاق النظام السحابي الأول"
   git branch -M main
   git remote add origin https://github.com/USERNAME/ae-storage.git  # استبدل USERNAME باسم حسابك
   git push -u origin main
   ```
