
# نظام A-E Storage Ecosystem - الدليل الشامل

هذا النظام هو بيئة عمل إدارية ومحاسبية متكاملة، مصممة لإدارة المستودعات، الديون، والعمليات المالية المعقدة بسلاسة تامة ومزامنة سحابية لحظية.

## 🚀 المواصفات التقنية (Tech Stack)
- **الإطار البرمجي:** Next.js 15 (App Router).
- **قاعدة البيانات:** Google Firebase Firestore (سحابية مجانية).
- **التصميم:** Tailwind CSS مع مكونات Shadcn UI الاحترافية.
- **الأمان:** نظام صلاحيات (Admin, Editor, Logger) مع رمز حماية إداري (Master Key).

## 📂 الوحدات والأنظمة المدمجة

### 1. وحدة إدارة المخزون (Inventory Management)
- **الأقسام (Departments):** تنظيم المواد في مجموعات رئيسية (مثلاً: مواد كهربائية، قطع غيار).
- **وحدات القياس (Units):** إدارة وحدات التعبئة (كرتونة، كجم، قطعة).
- **المخزن الذكي:** إضافة وتعديل المواد مع تتبع تلقائي للكميات، أسعار الشراء، وأسعار البيع.
- **تنبيهات النواقص:** نظام ذكي ينبهك عند وصول أي صنف لـ "حد الطلب" (أقل من 10 قطع).

### 2. وحدة الموردين والديون (Supplier & Debt System)
- **ملفات الموردين:** سجل كامل لكل مورد يشمل بيانات الاتصال والعنوان.
- **إدارة المديونيات:** حساب تلقائي لإجمالي المشتريات، المبالغ المسددة، والديون المتبقية لكل مورد.
- **سداد الديون:** واجهة مخصصة لتسجيل الدفعات المالية للموردين مع توثيق طريقة الدفع (كاش، تحويل، شيك).

### 3. وحدة المشتريات وفواتير التوريد (Purchasing System)
- **فواتير الشراء:** تسجيل فواتير توريد بضاعة مع ربطها تلقائياً بالمخزن لزيادة الكميات، وبالمورد لزيادة الديون.
- **تتبع الحالة المالية:** تصنيف الفواتير (مسددة بالكامل، مسددة جزئياً، غير مسددة).

### 4. وحدة الحوالات والكاش (Cash Transactions)
- **إدارة الحوالات:** سجل خاص بالمبالغ المرسلة والمستلمة من الأشخاص.
- **محرك البحث المالي:** ابحث باسم أي شخص ليظهر لك فوراً صافي حسابه (ما له وما عليه) بناءً على جميع حوالاته.

### 5. وحدة المالية العامة والمبيعات (General Finance)
- **المبيعات اليومية:** سجل لإجمالي عدد الفواتير اليومية وقيمة المبيعات العامة.
- **المصاريف:** تتبع تكاليف التشغيل (رواتب، إيجار، كهرباء) وتأثيرها على صافي الربح.
- **سجل الحركات (Movements):** أرشيف لا يُحذف يوثق كل حركة (وارد/صرف) بالوقت، التاريخ، واسم الموظف المسؤول.

### 6. التقارير والتحليلات (Reports & Analytics)
- **رسوم بيانية تفاعلية:** تحليل المنتجات الأكثر حركة (يومي، أسبوعي، شهري).
- **توزيع الأصول:** تمثيل مرئي لقيمة البضاعة في المخزن مقابل الديون الخارجية.

### 7. تصدير البيانات والنسخ الاحتياطي (Data Export)
- **Excel:** استخراج تقارير منظمة جداً تشمل 7 صفحات عمل (Sheets) كاملة البيانات.
- **PDF:** طباعة التقارير المالية والتحليلية.
- **JSON:** نسخة برمجية شاملة لكامل قاعدة البيانات (محمية برمز Master Key).
