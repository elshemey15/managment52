
# 🚀 نظام A-E Storage Ecosystem - دليل الحفظ السحابي والرفع

هذا النظام مصمم ليكون **خزنة بياناتك الدائمة**. البيانات محفوظة في سحابة جوجل (Firebase)، والكود الخاص بك سيتم حفظه في GitHub أو Netlify.

## 🔐 ضمان حفظ البيانات للأبد
- **لا حاجة للضغط على "حفظ":** النظام يستخدم تقنية المزامنة اللحظية مع **Firebase Firestore**.
- **الأمان:** حتى لو تعطل جهازك، بمجرد فتح الرابط وتسجيل الدخول، ستظهر بياناتك كاملة.

## 🛠 كيفية الرفع على Netlify (الأسرع والأسهل)
1. قم بإنشاء حساب مجاني على [Netlify](https://www.netlify.com/).
2. قم بسحب وإسقاط مجلد المشروع بالكامل في منطقة "Deploy manually".
3. أو قم بربط حسابك في GitHub وسيقوم Netlify بالبناء والنشر تلقائياً عند كل تحديث للكود.
4. الإعدادات جاهزة في ملف `netlify.toml` لضمان عمل الروابط العربية وبدون أخطاء.

## 💻 كيفية الرفع على GitHub
نفذ الأوامر التالية من الـ CMD داخل مجلد المشروع:
```bash
git init
git add .
git commit -m "إطلاق النظام السحابي الأول"
git branch -M main
git remote add origin https://github.com/USERNAME/ae-storage.git
git push -u origin main
```
