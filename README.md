
# 🚀 نظام A-E Storage Ecosystem - دليل الحفظ والتحميل

هذا النظام مصمم ليكون **خزنة بياناتك الدائمة**. البيانات محفوظة في سحابة جوجل (Firebase)، والكود الخاص بك يمكنك الآن تحميله وتشغيله على جهازك الشخصي.

## 📥 كيف تقوم بتحميل ملفات المشروع إلى جهازك؟
بما أنك تريد نقل الملفات إلى المسار `C:\Users\Administrator` على جهازك، اتبع الآتي:

1. **التحميل من المتصفح:** ابحث في واجهة **Firebase Studio** (الواجهة التي تراها الآن) عن أيقونة **"Download Project"** أو **"Export"** (عادة ما تكون على شكل سهم لأسفل أو صندوق). اضغط عليها لتحميل ملف `zip` يحتوي على كل الأكواد.
2. **فك الضغط:** قم بفك ضغط الملف في المسار التالي على جهازك:
   `C:\Users\Administrator\ae-storage`
3. **تجهيز البيئة:** افتح برنامج **CMD** (موجه الأوامر) في جهازك وانتقل للمجلد:
   ```bash
   cd C:\Users\Administrator\ae-storage
   ```
4. **التشغيل لأول مرة:** نفذ الأوامر التالية بالترتيب:
   ```bash
   npm install
   npm run dev
   ```
5. **الوصول للموقع:** افتح المتصفح واكتب: `http://localhost:3000`

## 🔐 ضمان حفظ البيانات للأبد
- **البيانات سحابية:** لا تقلق، حتى لو حذفت الملفات من جهازك، البيانات محفوظة في **Firebase Firestore**. بمجرد ربط الكود بملف الإعدادات الخاص بك، ستظهر بياناتك فوراً.
- **الأمان:** النظام يستخدم مفتاح رئيسي (Master Key) لحمايتك: `abdallah123456a`.

## 🛠 الرفع على GitHub أو Netlify
إذا كنت تريد رفع مشروعك أونلاين بعد تحميله:
- **Netlify:** اسحب مجلد `out` (الذي ينتج بعد أمر `npm run build`) وأسقطه في موقع Netlify.
- **GitHub:** استخدم أوامر `git` المعتادة لرفع المجلد الرئيسي.
