
# 🚀 دليل نظام A-E Storage Ecosystem

## 🔐 ضمان حفظ البيانات للأبد
النظام مرتبط بـ **Firebase Firestore**، وهي قاعدة بيانات سحابية من جوجل.
- **لا حاجة للضغط على "حفظ":** كل حركة تقوم بها تُحفظ فوراً في السحاب.
- **أمان البيانات:** بياناتك محفوظة على خوادم جوجل المشفرة، حتى لو ضاع جهازك أو قمت بمسح المتصفح، ستجد بياناتك بمجرد تسجيل الدخول.
- **الوصول من أي مكان:** يمكنك فتح الرابط من هاتفك أو جهاز الكمبيوتر في أي وقت وستظهر نفس البيانات المحدثة.

## 🛠 كيفية التشغيل من جهازك الشخصي
1. قم بإنشاء مجلد في المسار: `C:\Users\Administrator\ae-storage`.
2. انقل ملفات المشروع إليه.
3. افتح الـ CMD ونفذ:
   ```bash
   cd C:\Users\Administrator\ae-storage
   npm install
   npm run deploy
   ```
