
# 🚀 نظام A-E Storage Ecosystem - دليل الحفظ والتشغيل المحلي

هذا النظام مصمم ليكون **خزنة بياناتك الدائمة**. البيانات محفوظة في سحابة جوجل (Firebase)، والكود الخاص بك يمكنك الآن تحميله وتشغيله على جهازك الشخصي.

## 📥 كيف تقوم بتحميل الملفات إلى جهازك؟ (خطوة بخطوة)

1. **التحميل من المتصفح:** 
   - ابحث في شريط الأدوات العلوي في هذه الشاشة (Firebase Studio) عن زر **"Download"** أو **"Export"** (غالباً أيقونة سهم لأسفل).
   - سيتم تحميل ملف اسمه `project.zip` أو `ae-storage-ecosystem.zip`.

2. **نقل الملفات للمسار المطلوب:**
   - اذهب إلى مجلد التحميلات (Downloads) في جهازك.
   - اضغط بيمين الماوس على الملف واختر **"Extract All"** (فك الضغط).
   - قم بنقل المجلد الناتج إلى هذا المسار بالضبط:
     `C:\Users\Administrator\ae-storage`

3. **التشغيل على جهازك:**
   - افتح قائمة ابدأ (Start) واكتب **CMD** وافتحه.
   - اكتب الأمر التالي للانتقال للمجلد:
     ```bash
     cd C:\Users\Administrator\ae-storage
     ```
   - نفذ الأوامر التالية بالترتيب (تحتاج وجود Node.js مثبت على جهازك):
     ```bash
     npm install
     npm run dev
     ```
   - الآن افتح المتصفح واكتب: `http://localhost:3000` وستجد نظامك يعمل.

## 🔐 ضمان حفظ البيانات للأبد
- **لا تقلق على بياناتك:** حتى لو قمت بحذف الملفات من جهازك أو فورمات الويندوز، البيانات محفوظة في **Firebase Firestore**. بمجرد فتح الموقع مرة أخرى، ستظهر بياناتك فوراً لأنها "سحابية".
- **الأمان:** النظام يستخدم مفتاح رئيسي (Master Key) لحمايتك: `abdallah123456a`.

## 🛠 الرفع على GitHub أو Netlify
- **GitHub:** لرفع الكود، قم بإنشاء مستودع جديد (Repository) وارفع محتويات المجلد إليه.
- **Netlify:** اسحب مجلد `out` وأسقطه في موقع Netlify للنشر المباشر.
