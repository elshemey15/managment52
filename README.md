
# 🚀 دليل نقل نظام A-E Storage لجهازك (C:\Users\Administrator)

بما أنك تريد تشغيل النظام من جهازك الشخصي، اتبع هذه الخطوات بدقة:

## 1. نقل الملفات
* لا تضع الملفات مباشرة في مجلد `Administrator`.
* قم بإنشاء مجلد جديد اسمه `ae-storage` داخل `Administrator`.
* انقل جميع ملفات المشروع التي قمت بتحميلها من هنا إلى هذا المجلد: `C:\Users\Administrator\ae-storage`.

## 2. النشر من الجهاز (Deploy)
بعد نقل الملفات، افتح الـ CMD ونفذ هذه الأوامر بالترتيب:

1. **الدخول للمجلد (خطوة هامة جداً):**
   `cd C:\Users\Administrator\ae-storage`

2. **تثبيت المتطلبات (لأول مرة فقط):**
   `npm install`

3. **النشر النهائي (هذا الأمر سيقوم بالبناء والرفع في خطوة واحدة):**
   `npm run deploy`

## 3. حل مشكلة الصفحة الترحيبية
إذا استمرت الصفحة الترحيبية في الظهور بعد الـ Deploy:
1. تأكد من حذف ملف اسمه `index.html` (إذا وجد) داخل مجلد `public` في مشروعك قبل الرفع.
2. قم بعمل **Ctrl + F5** في المتصفح لمسح الذاكرة المؤقتة.
