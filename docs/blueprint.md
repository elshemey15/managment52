# **App Name**: تدفق المستودع

## Core Features:

- User Accounts & Permissions: Secure login system with distinct roles: 'System Admin' (full access), 'Editor' (add, modify, log movements, no delete), 'Logger' (only record movements, no modification/deletion/reporting).
- Category Management: UI for adding and editing product categories for organized inventory.
- Item Management: Interface to add, view, and modify product items with attributes like auto/manual code, name, category, unit, purchase price, sale price, and current quantity.
- Smart Inventory Search: Robust search and filtering functionality for items by name, code, or category, displaying current stock levels.
- Stock In/Out Transactions: Screen for recording inbound (وارد) and outbound (صادر) item movements, automatically capturing transaction date, time, and the user who made the entry. Updates inventory quantities instantly.
- Debt Account Management: Manages 'Supplier and Customer' (credit accounts) debts, automatically adjusting debt based on purchase (وارد) and sale (صادر) transactions.
- General Expenses & Financial Report: Dedicated interface to record general business expenses (e.g., rent, electricity) with description, date, and amount. Provides a financial report detailing total debts and specific account statements.

## Style Guidelines:

- The application will utilize a light color scheme for clarity and a professional feel, suitable for business management.
- Primary color: A medium corporate blue (#336699) to convey stability, reliability, and trust, ideal for financial and inventory management applications.
- Background color: A very light blue-grey (#E6ECF2), providing a clean and subtle canvas that enhances readability and keeps focus on the content.
- Accent color: A warm, inviting orange-yellow (#F2AE33) to highlight calls to action and important interactive elements, providing clear contrast and evoking activity.
- Body and headline font: 'Inter' (sans-serif) will be used consistently for all text. Its modern, neutral design ensures excellent readability for detailed reports, forms, and navigation across the application.
- Use clear, functional line icons that intuitively represent core actions (e.g., add, edit, delete, search) and navigation elements (e.g., inventory, accounts, reports), ensuring consistency across the application.
- A well-organized layout featuring a persistent sidebar for primary navigation. Data will be presented in structured tables and forms will follow a logical flow to optimize user input and information display.
- Incorporate subtle animations for interactive elements, such as hover effects on buttons and discreet loading indicators, to provide engaging user feedback without distracting from critical data management tasks.
